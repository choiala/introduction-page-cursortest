# 자기소개 페이지

김민수님의 포트폴리오 자기소개 페이지입니다.

## 파일 구조

- `introduction.html` - 메인 HTML 파일
- `introduction.css` - 스타일시트
- `introduction.js` - JavaScript 기능
- `introduceweb/P.png` - 프로필 이미지

## GitHub Pages 배포 방법

1. GitHub에서 새 저장소를 생성합니다
2. 다음 명령어로 파일을 푸시합니다:

```bash
git add .
git commit -m "Initial commit: 자기소개 페이지"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

3. GitHub 저장소의 **Settings** → **Pages** 메뉴로 이동
4. **Source**에서 `main` 브랜치와 `/ (root)` 폴더를 선택
5. 저장 후 몇 분 후에 `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME` 형태로 접속 가능합니다
